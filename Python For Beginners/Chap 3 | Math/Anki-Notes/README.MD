# Math 

##  Arithmatic Operators

| Arithmatic Operator   | What it does ?                                               |
| --------------------- | ------------------------------------------------------------ |
| `//` (Floor division) | divides the first operand by the second and the result is always an integer <br />print(7//2) => 3 |
| %(Modulas)            | Returns the remainder of the division of the first operand by the second <br />print(7%2) => 1 |
| `**` (Exponentiation) | Raises the first operand to the power of the second operand <br />print(7**2) => 49 |
| `/` (Division)        | Divides the first operand by the second .The result is always a float <br />print(7/2) => 3.5 |

## Boolean Operators 

| Boolean Operators | What they do ?                                               |
| ----------------- | ------------------------------------------------------------ |
| Boolean `or`      | returns True if at least one of the operands is `True`       |
| Boolean `and`     | returns True if all the operands are True                    |
| Boolean `not`     | inverts the value of the operand , if True then False, if False then True |
